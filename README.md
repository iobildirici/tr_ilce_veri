# tr_ilce_veri
İlçe düzeyinde Türkiye idari bölünüş verileri küçük ölçekli tematik harita yapımı için düzenlenmiştir.
Nokta, çizgi ve alan katmanları paylaşılmıştır. 
Veriler EPSG4326 koordinat sisteminde ve geopackage veri formatındadır.
Geopackage veritabanı üç katman (nokta çizgi alanı) içerir. 

Verilerde bir hata düzeltmesi nedeniyle veritabanı (gpkg dosyası) yeniden yüklenmiştir. (15.03.2025)

QGIS ile harita yapmak için ek dosyalar: (17.03.2026)
TR_LAMBERT.txt dosyası Türkiye haritaları için parametreleri düzenlemiş Lambert Konform Konik projeksiyonunun WKT formatında tanımını içerir.
idari.qml İdari sınırlar için QGIS'te kullanılabilecek çizgi tipleri için style dosyası
